# Steps

Write a function that accepts an integer and returns a 'staircase' of steps.

For example:

`stepper(6);`

... returns

```
[ [ '#', ' ', ' ', ' ', ' ', ' ' ],
  [ '#', '#', ' ', ' ', ' ', ' ' ],
  [ '#', '#', '#', ' ', ' ', ' ' ],
  [ '#', '#', '#', '#', ' ', ' ' ],
  [ '#', '#', '#', '#', '#', ' ' ],
  [ '#', '#', '#', '#', '#', '#' ] ]
```


# Bonus

Make it recursive.
